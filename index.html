Projects

Last year, I joined a hackathon and hacked up a Ruby on Rails app, the PLEASED app for a medical researcher to use in her study, with an amazing and enthusastic team. I was also a student dev on the Markus open source project, a Rails online marking app. In both experiences, while I was able to contribute, I was limited by my novice knowledge of the Ruby language and of the Rails framework, I was not able to contribute as much as I had hoped to. With Rails, it's quick to look up a tutorial and get started with a twitter clone app really fast, but when working on a real codebase, it's easy to get lost. That's why I'm working through the Odin Project lessons to gain a deeper understanding of Ruby.

Currently I'm working on two mobile applications, Agents of Nature (an educational game for kids to play outdoors) and Explore150 (an app for discovering Canadian natural, historical and cultural sites). We build the apps using a cross-platform C++ sdk, Marmalade to target both iOS and Android with one C++ codebase. I'd really like to experiment with native Java Android apps, and with Apache Cordova to see how mobile apps are more commonly created. I've only gone through and created the most basic apps of each type, thus far.
